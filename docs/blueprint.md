# **App Name**: Complaint Central

## Core Features:

- Report Display: Display detailed complaint reports including incident photos, vehicle details (type, plate number), route, time, and date.
- Report Filtering: Enable filtering and searching of reports based on vehicle type, date, and other relevant criteria.
- Resolve Reports: Provide the option to mark reports as 'resolved' with a visual indicator.
- New Report Upload: A system for uploading new complaint reports, ensuring that required fields (e.g., incident photo, vehicle type, date, time, description) are completed. When insufficient details are provided, use AI to request further clarification from the user.
- AI Complaint Classification: AI-powered tool that uses incident photos to classify the complaint type.
- Download Reports: Ability to download reports in various formats for offline analysis and record-keeping.

## Style Guidelines:

- Primary color: (#557FFB)
- Background color:  (#FFFFFF)
- Accent color: Teal (#45A848)
- Body and headline font: 'Inter', a grotesque-style sans-serif font for a clean and modern look.
- Use simple, clear icons to represent vehicle types and report statuses.
- Implement a grid-based layout for consistent alignment and easy scanning of report data.
- Use subtle transitions and animations when filtering or marking reports as resolved.
