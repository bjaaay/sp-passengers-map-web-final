Source layout

- `app/` — Next.js app routes and pages (Next 13+ app router).
- `components/` — Shared UI components and small UI primitives (see `components/ui`).
- `hooks/` — Custom React hooks used across the app.
- `lib/` — Utilities, firebase setup, types and small helper modules.

Quick start

1. Install dependencies:

```powershell
npm install
```

2. Run dev server (the project runs on port 9002):

```powershell
npm run dev
```
